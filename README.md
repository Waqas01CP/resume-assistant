An AI-powered assistant to generate personalized cover letters and resume improvements

# Resume & Cover Letter Assistant

This Jupyter Notebook helps users generate:
- A customized **cover letter** based on your resume and a job description
- Personalized **resume improvement suggestions** using OpenAI GPT

---

## How It Works

1. Paste your **OpenAI API key** when prompted
2. Paste your **resume**
3. Paste the **job description**
4. Click run — and let the AI do the rest 

You’ll get:
- A tailored **cover letter**
- Direct, actionable suggestions to improve your resume

---

## Requirements

This notebook runs best in **Google Colab** or **Jupyter Notebook**.  
If you're running it locally, install the required packages below.

### Install Dependencies (for local use)

```bash
pip install openai python-dotenv
```

##API Key Note
When the notebook starts, you’ll be prompted to paste your OpenAI API key manually.
No .env file is needed. Your key is not stored anywhere.
